This mod edits some of the Vanilla Recipes and
adds some of its own to add more mod support


Recipes Changed

Hoppers
replaced Item minecraft:chest
with Tag c:wooden_chests
Shulker Boxers
replaced Item minecraft:chest
with Tag c:wooden_chests
Boat with Chests
replaced Item minecraft:chest
with Tag c:wooden_chests


Recipes Added

Wood Fuel
burn time 300 ticks / 1.5 items
uses Tag quad:wood_fuel


Tags allow multiple possible items to be used in a Recipe instead of being fixed to 1 item.
Tags are also easily changed by other mods, meaning any mod can use these features

Tags with `fuel` in allow the items to burn in a furnace for the
amount of ticks displayed - and amount of items that can be cooked

Know a mod that could benefit from Quad? Join my discord for support requests.
